## 🚀 快速开始

### 1. 环境要求

- Python 3.8+
- 现代浏览器（Chrome、Firefox、Edge等）

### 2. 安装依赖

```bash
# 克隆项目
git clone <repository-url>
cd 教学平台

# 创建虚拟环境（推荐）
python -m venv venv

# 激活虚拟环境
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 3. 启动平台

```bash
python app.py
```

访问 http://localhost:5000 开始学习之旅！
