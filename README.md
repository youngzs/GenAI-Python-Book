# Python智能体开发教材 - 项目总览

## 🎯 项目简介

这是一个全面的Python智能体开发教材项目，采用PDCA (Plan-Do-Check-Act) 方法论进行编写管理。教材分为三册，共52章，从Python基础到AI智能体开发的完整学习路径。
